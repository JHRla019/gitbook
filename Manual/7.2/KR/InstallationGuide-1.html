제 1장. InnoRules 설치

1. 설치 전 준비

InnoRules의 소프트웨어를 설치하기 위해서는 사전에 확인이 필요한 준비물은 아래와 같다.

  • InnoRules의 Setup 파일
  • InnoRules를 사용하기 위한 사용자 PC 사양
사양 최소사양 권장 사양
OS MS Windows 7 이상 MS Windows 7 이상
CPU 1.6 GHz 2Core * 1 2.4 GHz 2Core * 1 이상
Main Memory 2GB 4GB 이상
HDD 100MB 이상 100MB 이상
화면 해상도 1600Ｘ900 1600Ｘ900 이상

2. InnoRules 설치

InnoRules 설치는 아래의 단계에 따라 설치한다.

  1. InnoRules Setup 파일을 실행한다. “설치 언어 선택 창”이 나타난다.

    [그림1. 설치 언어 선택 창]

    설치 과정에서 사용할 언어를 변경하려면 화면의 아래 화살표를 클릭하여 설치 시 사용할 언어를 변경한다. (이 언어의 선택은 InnoRules에서 사용할 언어와는 관계가 없다. InnoRules에서 사용할 언어는 현재 Windows에 설정된 국가 언어에 따라 자동으로 설정된다.)

    InnoRules의 설치를 원하면 OK 버튼을 클릭하여 다음으로 진행 한다. 만일 설치를 원하지 않으면 Cancel 버튼을 클릭하여 설치 프로그램을 종료한다.

  2. 설치 안내 문구 화면

    [그림2. 설치 안내 문구 화면]

    설치를 시작하기 전에 실행중인 InnoRules가 있다면 종료해야 한다. 실행중인 InnoRules가 있으면 설치가 진행되지 않는다.

    설치를 원하면 다음 버튼을 클릭하여 “4. 설치 위치 선택” 화면으로 진행한다. 설치를 원하지 않으면 취소 버튼을 클릭하여 “3. 설치 취소 창” 화면으로 진행한다.
